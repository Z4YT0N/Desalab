Getting Started

The Solar Desalination Simulator is a powerful tool for analyzing and optimizing solar desalination systems. Follow these steps to get started:

  1. Navigate to the Simulator page
  2. Enter your weight measurements at 5-minute intervals
  3. Input corresponding temperature readings
  4. Select your absorber shape and specify dimensions
  5. Adjust the factor value if needed
  6. Click "Run Simulation" to see your results

Input Parameters

Weight Measurements

Enter weight values in grams (g) at 5-minute intervals. The system expects 13 measurements, starting from time 0 to 60 minutes. Values should be decreasing over time.

Temperature Records

Input temperature readings in Celsius (°C) corresponding to each time interval. Values should be between 0°C and 100°C.

Absorber Parameters

Available Shapes:

  • Circle (requires radius)
  • Square (requires side length)
  • Rectangle (requires length and width)
  • Triangle (requires base and height)

All dimensions should be entered in centimeters (cm).

Calculations

Mass Loss

Mass loss is calculated as the difference between initial and current weight measurements. Results are provided in various units: grams (g), kilograms (kg), and kg/m².

Mass Flux

Mass flux (g/m²·s) is calculated using the total mass loss per unit area over the experiment duration.

Efficiency

Efficiency calculation considers:

  • Specific heat capacity of water (4.18 J/g°C)
  • Latent heat of vaporization (2400 J/g)
  • Temperature difference between surface and bulk
  • Mass flux

Results

Visualization

Results are presented through:

  • Interactive plots showing mass loss over time
  • Mass loss per square meter visualization
  • Comprehensive results table with all measurements
  • Efficiency and mass flux indicators

Data Export

All results can be exported for further analysis or reporting purposes.

Troubleshooting

Common Issues

Invalid Input Data

Ensure all fields contain valid numeric values and follow the specified ranges.

Dimension Errors

Verify that all required dimensions are provided for the selected absorber shape.

Calculation Errors

Check that weight values are decreasing over time and temperatures are within reasonable ranges.

API Reference

Endpoints

POST /simulate

Run a simulation with provided parameters. 

{
    "weights": [float],      // Array of 13 weight measurements
    "temperatures": [float], // Array of 13 temperature readings
    "absorberShape": string, // "circle", "square", "rectangle", "triangle"
    "dimension1": float,     // First dimension in cm
    "dimension2": float,     // Second dimension in cm (if applicable)
    "factor": float         // Adjustment factor
}
